Zero-Server Infrastructure

How this portfolio is hosted

This portfolio demonstrates a Zero-Server Architecture — a secure, cost-effective hosting solution that repurposes an Android device as an edge server instead of using traditional VPS hosting.

$0/month

Hosting Cost

Zero Ports

Public Exposure

ARM64

Low Power

Architecture Components

  • Host: Android (ARM64) using Termux
  • Web Server: Nginx
  • Security: Cloudflare Zero Trust (encrypted tunnel)
  • Deployment: GitHub Actions (CI/CD)
  • Domain: kunjidev.shop

How It Works

Traffic flows from visitors → Cloudflare Edge → Encrypted Tunnel → Android Device → Nginx. This provides SSL termination, DDoS protection, and complete IP obfuscation without requiring port forwarding or exposing the device to the public internet.

Key Benefits

  • No public IP exposure or inbound ports open
  • Automated deployment through GitHub Actions
  • Enterprise-grade security via Cloudflare
  • Real DevOps & infrastructure automation experience
